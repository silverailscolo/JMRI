The StringIO Table

You can use the StringIO Table to look at the contents of JMRI StringIO objects.

Note that you can sort the table by each of the columns, just click on the column heading.

To change the value stored in a particular StringIO, just double click on the Value column in the desired row. Enter a new value, followed by hitting the Return or Enter key to store the string value. For system-specific StringIOs, this will send the value out to the layout immediately
Entering an empty value will store an empty String value to the variable.

Adding a new StringIO

To create a new StringIO, click on the "Add..." button at the bottom left of the pane.

For more information on StringIOs, see the StringIOs help page.
